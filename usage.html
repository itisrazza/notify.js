How to use Notify.js

To use Notify.js, you first need to link the notify.min.js and the CSS file for your theme (Element is used as an example). If you use Element, it should look like this...

<link rel="stylesheet" type="text/css" href="assets/notify/element.css">
<script type="text/javascript" src="demo/notify.js"></script>

After you've done that, go down to the end of the page and if you are using Element, type this:

<div id="notifyCloak" class="" onclick="notifyHide()"></div>
<div class="notify-box notify-hide" id="notifyJs">
  <a onclick="notifyHide()" class="notify-close">&times;</a>
  <div class="notify-img" id="notifyIcon"></div>
  <div class="notify-content">
    <h4 id="notifyTitle">You haven't set anything.</h4>
    <p id="notifyContent">You should set something before doing it again.</p>
  </div>
</div>

And you should be ready to go.

Summoning the notification

To set the notification text, push the following command: notifySet('Title','Content','Icon location',false);. Feel free to change the attributes above. You can set the last attribute to true to make the notification show right after it gets set.

You can make the notification disappear by using notifyHide(); and appear yet again with notifyShow();. A toggle is also available if you want that... just use notifyToggle();.
